# Zscaler Installer

Batch script for deploying Zscaler security software.

## Description

Remotely installs Zscaler security client on target computers using PsExec.

## Usage

```batch
:: Multiple computers
1.bat
```

## Files

- `1.bat` - Batch runner
- `Zscaler.bat` - Main installer script
- `computers.txt` - Target computers
- `success.txt` - Successful installations
- `Error.txt` - Failed installations

## Requirements

- Windows OS
- Administrative privileges
- PsExec
- Zscaler installation package
