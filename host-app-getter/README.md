# Host App Getter

Batch script to retrieve installed applications from a specific host.

## Description

Uses WMIC to query and display installed applications from a target computer in HTML format.

## Usage

```batch
:: Run the script
App.bat
```

## Output

Generates HTML file with:
- Application name
- Version
- Vendor

## Files

- `App.bat` - Main script

## Requirements

- Windows OS
- WMIC access
- Network connectivity
