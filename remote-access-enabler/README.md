# Remote Access Enabler

Batch scripts to enable remote access features on computers.

## Description

Enables remote management capabilities on target computers including:
- Remote Desktop
- WinRM
- PowerShell Remoting

## Usage

```batch
:: Run on target computers
1.bat
```

## Files

- `1.bat` - Batch runner
- `Local_account_Script.bat` - Main enablement script
- `computers.txt` - Target computer list

## Requirements

- Windows OS
- Administrative privileges
- PsExec (PsTools)
