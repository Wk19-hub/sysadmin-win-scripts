# User Deleter

Scripts for deleting user profiles from computers.

## Description

Removes specific user profiles from local or remote computers.

## Usage

```batch
:: Run deletion scripts
1.bat
```

## Files

- `Deleting users/` - User deletion scripts

## Warning

⚠️ This permanently deletes user data. Ensure backups exist.

## Requirements

- Windows OS
- Administrative privileges
- delprof2 or similar tool
