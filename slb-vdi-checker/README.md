# SLB VDI Checker

Scripts for checking VDI (Virtual Desktop Infrastructure) status.

## Description

Verifies VDI machine status and connectivity.

## Usage

```batch
:: Run checks
1.bat
```

## Files

- `1.bat` - Batch runner
- `Script.bat` - Main check script
- `com.txt` - Target VDI machines

## Requirements

- Windows OS
- Network connectivity to VDI infrastructure
