# Edge Installer

Batch script for remote deployment of Microsoft Edge browser.

## Description

Remotely installs Microsoft Edge on multiple computers using batch scripts.

## Usage

```batch
:: Run batch installer
1.bat
```

## Files

- `1.bat` - Batch runner for multiple computers
- `edge.bat` - Main installation script
- `computers.txt` - List of target computers

## Requirements

- Windows OS
- Administrative privileges on targets
- Edge installer package
