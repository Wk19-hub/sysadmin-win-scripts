# SCCM Troubleshooter 2

Additional SCCM troubleshooting scripts.

## Description

Extended collection of SCCM client troubleshooting scripts for more complex issues.

## Usage

```batch
1.bat
```

## Files

- `1.bat` - Batch runner
- `Local_account_Script.bat` - Main script
- `Local_account.bat` - Additional script
- `computers.txt` - Target computers

## Requirements

- Windows OS
- PsExec (PsTools)
- Administrative privileges
