# Smart Client Installer

Batch scripts for deploying smart client applications.

## Description

Remote deployment of smart client software to multiple computers.

## Usage

```batch
:: Run deployment
1.bat
```

## Files

- `1.bat` - Batch runner
- `PH.bat` - Main installation script
- `computers.txt` - Target computers
- `smart/` - Installation files (add your own)

## Requirements

- Windows OS
- PsExec (PsTools)
- Administrative privileges
