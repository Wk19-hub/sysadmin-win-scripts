# File Permissions Checker

Batch script to check and manage file permissions on remote computers.

## Description

Verifies file and folder permissions on target computers and generates reports.

## Usage

```batch
:: Run checks
1.bat
```

## Files

- `1.bat` - Main script
- `computers.txt` - Target computers
- `Results.txt` - Output report
- `RetryList.txt` - Machines needing retry

## Requirements

- Windows OS
- Administrative privileges
- Network access to admin shares
