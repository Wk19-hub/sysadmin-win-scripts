# Old File Deleter

Batch script to delete old files and copy new ones.

## Description

Removes outdated files from target computers and replaces with new versions.

## Usage

```batch
:: Single computer
Copy_With_Ping.bat COMPUTERNAME

:: Multiple computers
1.bat
```

## Files

- `1.bat` - Batch runner
- `Copy_With_Ping.bat` - Main script
- `computers.txt` - Target computers

## Requirements

- Windows OS
- Administrative privileges
- Network access to admin shares
