# User Deleter / Users Deleter

Scripts for deleting user profiles from computers.

## Description

Removes user profiles from local or remote computers using delprof2 or similar tools.

## Usage

```batch
:: Delete specific user profile
ExpProfile.bat

:: Delete from multiple computers
1.bat
```

## Files

- `1.bat` - Batch runner
- `ExpProfile.bat` - Profile deletion script
- `PH.bat` - Additional script
- `ONLY ONE.bat` - Single profile deletion
- `DU/` - Additional deletion utilities

## Warning

⚠️ This permanently deletes user data. Ensure backups exist.

## Requirements

- Windows OS
- delprof2 or similar tool
- Administrative privileges
