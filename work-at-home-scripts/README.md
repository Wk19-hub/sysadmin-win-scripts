# Work-From-Home Scripts

Collection of scripts used for configuring and supporting work-from-home (WFH) environments. These are genericized for public release and have been anonymized.

## Description

Contains installers and configuration helpers for remote/telework setups (VPN, agent installs, network prerequisites).

## Usage

Navigate to the specific script and follow the README instructions inside subfolders.
