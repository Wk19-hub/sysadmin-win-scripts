# Sleep Mode Manager

Scripts to configure sleep and power settings on computers.

## Description

Configures power management settings including sleep mode, hibernation, and display timeout.

## Usage

```batch
:: Run configuration
1.bat
```

## Files

- `1.bat` - Batch runner
- `Local_account_Script.bat` - Main configuration script
- `Local_account.bat` - Additional script
- `computers.txt` - Target computers
- `local-security-policy.inf` - Security policy template

## Power Settings Configured

- Sleep timeout
- Display timeout
- Hibernation settings
- Wake timers

## Requirements

- Windows OS
- Administrative privileges
- PsExec (PsTools)
